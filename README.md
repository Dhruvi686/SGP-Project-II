# Tourist Help Hub - Ladakh

A full-stack **MERN** web application that provides information and assistance for tourists visiting **Ladakh**.  
This project consists of a **React frontend** and **Node/Express backend**, offering features such as destination info, local help resources, and interactive UI for users.
